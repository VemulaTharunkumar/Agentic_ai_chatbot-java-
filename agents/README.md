# 🚀 Agentic AI System  
### Autonomous Multi-Agent AI for Real-World Task Execution

Agentic AI System is an **autonomous multi-agent artificial intelligence platform** that can **research the internet, write and execute code, compile Java programs, train machine learning models, evaluate accuracy, debug errors, and fulfill user tasks end-to-end**.

This project demonstrates **advanced AI orchestration**, **LLM-powered agents**, and **real-world automation**, inspired by AutoGPT, Devin AI, and OpenAI Agents.

---

## 🔥 Key Features

- Multi-agent AI collaboration
- Real-time web research from live websites
- Python execution & Java compilation
- Machine learning model training & evaluation
- Automatic debugging & self-improvement
- Vector-based long-term memory
- API calling & dataset downloading
- Autonomous task orchestration
- Scalable & modular architecture

---

## 🧠 AI Agents

| Agent | Role |
|------|------|
Planner Agent | Task decomposition & strategy  
Research Agent | Live internet research  
Coder Agent | Writes executable code  
Python Executor | Runs Python programs  
Java Executor | Compiles & runs Java  
ML Agent | Trains & evaluates ML models  
Critic Agent | Reviews & improves output  
Memory Agent | Stores long-term knowledge  

---

## ⚙️ Tech Stack

- Python
- OpenAI API (LLMs)
